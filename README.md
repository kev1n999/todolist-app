# TodoList app 
A simple todolist app in nextjs

--- 

# Used Tools 
> - NodeJS
> - TypeScript
> - NextJS
> TailwindCSS

# How to use
1. Clone the repository 
```
git clone https://github.com/kev1n999/todolist-app/ 
```
2. Install the dependencies
```
npm install 
```
3. Create a .env 
```
NEXT_PUBLIC_SERVER_URL=http://localhost:8000/
```
