# TodoList app 
A simple todolist app in nextjs to use with **[todolist-api](https://github.com/kev1n999/todolist-api)**

--- 

# Used Tools 
> - NodeJS
> - TypeScript
> - NextJS
> TailwindCSS

# How to use
1. Clone the repository 
```
git clone https://github.com/kev1n999/todolist-app/ 
```
2. Install the dependencies
```
npm install 
```
3. Create a .env 
```
NEXT_PUBLIC_SERVER_URL=http://localhost:8000
``` 
4. Run the project 
```
npm run dev
```  

---

# Obs
> It just work if you're using the **[todolist-api](https://github.com/kev1n999/todolist-api)**
